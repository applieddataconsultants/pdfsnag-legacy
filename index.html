pdfsnag - streaming pdf generator

Base Request Params (GET and POST)

   url: url to generate pdf from
   html: html to generate pdf from
   name: name of generated pdf (default: output.pdf)

Additional wkhtmltopdf Request Params Supported

%OPTIONS%

Read about the options

Example: Sample GET Requests

   ?url=google.com
   ?html=<h1>Hi mom&to_mom
   ?url=google.com&zoom=3&orientation=Landscape&grayscale=&encoding=UTF-8&page-height=17in&page-width=11in
